# WezTerm Config

My personal WezTerm configuration. **macOS only.**

## Features

- Tokyo Night color scheme
- JetBrains Mono + CJK font fallback
- Pane management with `Alt` + hjkl / arrow keys
- **Equalize all panes** with `Alt` + `=` (binary split tree reconstruction algorithm)
- Manual pane resize with `Alt` + Shift + HJKL
- Tab switching with `Cmd` + number / jk
- Toggle window opacity with `Ctrl` + `Alt` + `o`
- Rename tab with `Cmd` + Shift + `R`
- Auto maximize on startup

## Keymaps

| Shortcut | Action |
|---|---|
| `Alt + n` / `Alt + Shift + N` | Split horizontal / vertical |
| `Alt + h/j/k/l` or Arrow keys | Switch pane |
| `Alt + Shift + H/J/K/L` | Resize pane |
| `Alt + =` | Equalize all panes |
| `Alt + x` | Close pane |
| `Cmd + n` | New tab |
| `Cmd + x` | Close tab |
| `Cmd + 1-9` | Switch to tab |
| `Cmd + j/k` | Previous / next tab |
| `Cmd + Shift + R` | Rename tab |
| `Ctrl + Alt + o` | Toggle opacity |
